CSS #3 Text Effects →

Text shadow property allow us to to butify the text like photoshop. text-shadow formula can be define as follow

text-shadow : horizontal-shadow[h-shadow] vertical-shadow[v-shadow] blur-radius shadow-color;

→ Horizontal offset of shadow( X-axis ). positive value show shadow to right side of the text & negative value show shadow to left side of the text.

→ Vertical offset of shadow( Y-axis ). positive value show shadow to below the text & negative value show shadow to labove the text.

→ This is optional. if left blank default vaule is set to 0, blur radius define the how much & how big the shadow should be blur.
